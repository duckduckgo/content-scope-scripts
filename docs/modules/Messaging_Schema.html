    Module Messaging Schema

    These are all the shared data types used throughout. Transports receive these types and can choose how to deliver the message to their respective native platforms.

    • Notifications via NotificationMessage
    • Request -> Response via RequestMessage and MessageResponse
    • Subscriptions via Subscription

    Note: For backwards compatibility, some platforms may alter the data shape within the transport.
