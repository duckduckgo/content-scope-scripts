    Module Duck Player Overlays

    Duck Player Overlays are either the small Dax icons that appear on top of video thumbnails when browsing YouTube. These icons allow users to open the video in Duck Player.

    On the YouTube player page, the main Duck Player Overlay also allows users to open the video in Duck Player, or dismiss the overlay.

    Messages:

    On Page Load

    • DuckPlayerOverlayMessages.initialSetup is initially called to get the current settings
    • DuckPlayerOverlayMessages.onUserValuesChanged subscription begins immediately - it will continue to listen for updates
    • DuckPlayerOverlayMessages.onUIValuesChanged subscription begins immediately - it will continue to listen for updates

    Then the following message can be sent at any time

    • DuckPlayerOverlayMessages.setUserValues
    • DuckPlayerOverlayMessages.openDuckPlayer

    Please see DuckPlayerOverlayMessages for the up-to-date list

    Remote Config

    • Please see OverlaysFeatureSettings for docs on the individual fields
